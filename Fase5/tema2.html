Atributos de HTML

Los atributos de HTML son palabras especiales utilizadas dentro de la etiqueta de apertura, para controlar el comportamiento del elemento. Los atributos de HTML son un modificador de un tipo de elemento de HTML. Un atributo tampoco modifica la funcionalidad por defecto de un tipo de elemento o proporciona funcionalidad a ciertos tipos de elementos incapaces de funcionar correctamente sin ellos.

La mayoría de los atributos de un elemento son pares de nombre y valor , separados por "="" y escritos dentro de la etiqueta de inicio de un elemento después del nombre del elemento. El valor se puede encerrar entre comillas simples o dobles, aunque los valores que consisten en ciertos caracteres se pueden dejar sin comillas en HTML. Dejar valores de atributo sin comillas se considera inseguro. A diferencia de los atributos de par nombre-valor, hay algunos atributos que afectan al elemento simplemente por su presencia en la etiqueta de inicio del elemento, como el "ismap" atributo del "img" elemento.


Hay varios atributos comunes que pueden aparecer en muchos elementos:

El atributo "id" proporciona un identificador único en todo el documento para un elemento. Esto se usa para identificar el elemento para que las hojas de estilo puedan alterar sus propiedades de presentación y los scripts puedan alterar, animar o eliminar su contenido o presentación. Agregado a la URL de la página, proporciona un identificador global único para el elemento, generalmente una subsección de la página.

El atributo "class" proporciona una forma de clasificar elementos similares. Esto se puede utilizar con fines semánticos o de presentación. Por ejemplo, un documento HTML podría usar semánticamente la designación para indicar que todos los elementos con este valor de clase están subordinados al texto principal del documento. En la presentación, estos elementos se pueden reunir y presentar como notas al pie de página en lugar de aparecer en el lugar donde aparecen en el código fuente HTML. Los atributos de clase se utilizan semánticamente en microformatos . Se pueden especificar varios valores de clase; por ejemplo , coloca el elemento en las clases y en las clases.<class="notation"><class="notation important">notation important

Un autor puede usar el "style" atributo para asignar propiedades de presentación a un elemento en particular. Se considera una mejor práctica usar un elemento "id" o "class" atributos para seleccionar el elemento dentro de una hoja de estilo , aunque a veces esto puede ser demasiado engorroso para un estilo simple, específico o ad hoc.

El atributo "title" se utiliza para adjuntar una explicación subtextual a un elemento. En la mayoría de los navegadores , este atributo se muestra como información sobre herramientas.

El atributo "lang" identifica el idioma natural del contenido del elemento, que puede ser diferente al del resto del documento.

Atributos globales



Los atributos globales, también conocidos como atributos estándares, funcionan con un gran número de elementos. Técnicamente todos los atributos globales tienen que ser aceptados por todos los elementos, aunque no funcionen con algunos de ellos
